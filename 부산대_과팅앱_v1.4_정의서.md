## 1. 앱 아이덴티티 확정

### 앱 이름: **Destiny** (운명)

v1.3까지 잠정 "부산대 과팅앱"으로 불렸으나 v1.4에서 공식 브랜드명으로 **Destiny** 확정.

## 2. v1.4에서 완성된 구현

### 2-1. Soul Orbs 매칭 풀 시각화 (`components/MatchingPool.tsx`)

랜딩 페이지에 삽입된 실시간 대기 인원 시각화 컴포넌트.

**구성 요소**
- 오브 10개 (여성 5개 = 로즈핑크, 남성 5개 = 바이올렛)
- 각 오브: aura glow + 본체 + 만남 flash 레이어 3중 구조
- 레이더 필드: 외부 glow 링 + 동심원 2개 + 회전 스캔 라인
- 카운터: 여성/남성/총계 카운트업 애니메이션 (1.2초, cubic ease)
- 실시간 뱃지: 녹색 펄스 점 + "실시간 업데이트" 텍스트

**CSS 애니메이션 (tailwind.config.ts keyframes)**

| keyframe | 용도 |
|----------|------|
| `orb-float-1~5` | 오브 유기적 부유 (각기 다른 경로) |
| `orb-pulse` | 오브 aura 호흡 |
| `radar-sweep` | 레이더 스캔 라인 회전 (10s) |
| `ring-expand` | 중심 펄스 링 확장 |

**구현 주의사항**  
Tailwind arbitrary value (`animate-[...]`)는 동적으로 생성한 클래스명에 작동하지 않음. 인라인 `style={{ animation: ... }}` 방식 사용.

### 2-2. DestinyLogo SVG 재설계

Manus 디자인 가이드의 버터플라이 오빗 + 4각별 로고 반영.

```
- 두 타원: rotate(-38°)과 rotate(38°), rx=38 ry=15, 교차점에 점(dot)
- 그라디언트: #f9a8d4 → #e879f9 → #7c3aed (로즈골드 → 바이올렛)
- 중심: 4각별 Path + 흰색 코어 + blur 글로우 레이어
```

### 2-3. 전체 페이지 디자인 시스템 통일

**모든 프로필 페이지 h1** → `gradient-fate-text` 클래스 적용  
(기본, 사진, 성격 테스트, 시간대, 가중치 페이지)

**이모지 → Lucide 아이콘 교체**

| 컴포넌트 | 교체된 아이콘 |

| `Big5Survey` | Palette / ClipboardCheck / Zap / Heart / Waves |

| `PreferenceSliders` | Sparkles / MessageCircle / Ruler / Dumbbell / GraduationCap / Gamepad2 / Clock |
| `SelfWorldcupResult` | Sparkles / Star / Crosshair |

**Manus 카피 반영**

## 5. 남은 주요 과제

### 즉시 해결 필요

1. **Supabase 실 키 연동**: `.env.local`에 `NEXT_PUBLIC_SUPABASE_URL`, `NEXT_PUBLIC_SUPABASE_ANON_KEY` 입력 → 로그인 실제 동작
2. **SCUT-FBP5500 모델 가중치**: Python 서버에 실 가중치 파일 로드
3. **Supabase Storage RLS**: `photos` 버킷 읽기/쓰기 정책 확인 (성준과 협의)

### 단기 (v1.5 목표)

4. **64강 자기유사 월드컵 이미지 교체**: Codex 결과물 수령 후 반영
5. **그룹 생성 화면** (`/group/create`): 성준 파트, 완성 후 연동
6. **매칭 대기 실시간**: MatchingPool 컴포넌트에 실 Supabase 실시간 구독 연결
7. **프로필 완성도 체크**: 미완성 상태에서 그룹 가입 시 /profile/edit 리다이렉트

### 중기 (v2.0 목표)

8. **보증금 결제**: 토스페이먼츠 테스트 키 연동
9. **매칭 엔진 연동**: 성준의 헝가리안 알고리즘 서버 API 연결
10. **만남 후 리뷰 시스템**: 노쇼/거짓말 신고 플로우
11. **PWA 알림**: 매칭 확정 시 푸시 알림
