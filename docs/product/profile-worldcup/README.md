# product/profile-worldcup

이상형 월드컵, 외모/성격 벡터, 외모 분석 문서. 이미지 메타데이터는 `image-metadata/` 하위에.

| 문서 | 내용 |
|---|---|
| `IDEAL_WORLDCUP_64_DESIGN.md` | 이상형 월드컵 64장 풀 설계 |
| `IDEAL_WORLDCUP_MEASURED_VECTOR_PLAN.md` | 측정 벡터 기반 월드컵 계획 |
| `APPEARANCE_ANALYSIS_SCHEMA.md` | 외모 분석 결과 스키마 |
| `APPEARANCE_ANALYSIS_GPT_PROMPT.md` | 외모 분석 GPT 프롬프트 |
| `APPEARANCE_VECTOR_CALIBRATION.md` | 외모 벡터 캘리브레이션 |
| `image-metadata/` | 남/여 64장 이미지 카탈로그·메타데이터 |

> 실제 점수 데이터: `public/appearance-ideal/METADATA.json`. 로더: `lib/appearance/metadata.ts`.
