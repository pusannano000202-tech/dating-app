# delete-candidates

삭제해도 될 것 같지만 **바로 지우지 않고** 1회 검토 주기 동안 격리하는 폴더.

## 규칙

- 첫 정리 패스에서는 직접 삭제하지 않는다.
- 의심스러운 파일을 여기로 옮긴다.
- 아래 표에 **왜 옮겼는지** 사유를 적는다.
- 한 번 검토 주기가 지난 뒤, 확실히 폐기된 파일만 별도 커밋으로 삭제한다.

## 격리 목록

| 파일 | 옮긴 사유 | 격리일 |
|---|---|---|
| (없음) | 1차 정리에서는 보존 우선 — 폐기 확정 파일 없음. 옛 문서는 `../archive/`로 보존 이동함 | — |

> 현재 비어 있음. 폐기 확정 시 이 표에 기록 후 이동.
