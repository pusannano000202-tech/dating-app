# Plans Hub

이 폴더는 프로젝트 계획서와 현재 의사결정 문서를 한곳에서 찾기 위한 허브다. 기존 `docs/`와 `docs/handoff/` 문서는 보존하고, 여기서는 현재 기준의 실행 계획과 참조 인덱스를 관리한다.

## 운영 규칙

- 새 기능 방향이 정해지면 이 폴더에 날짜별 설계/계획 문서를 먼저 만든다.
- 기존 문서가 흩어져 있어도 파일 이동은 신중하게 한다. 우선 이 폴더에서 링크와 요약으로 관리한다.
- 구현이 끝난 계획은 문서 상단에 결과와 관련 커밋/PR을 추가한다.
