# 여자 자기유사 월드컵 시범 이미지 메타데이터

점수는 백분위 절대점수다. 생성 프롬프트의 목표 점수와 실제 판정 점수가 다르면 실제 판정 점수를 우선한다.

## `female_self_28.jpg`

- 최종 외모절대점수: 28
- 백분위 의미: 하위 20~30% 구간
- 원래 생성 목표: 25점대
- 판정: 목표와 대체로 일치
- 핵심 신호: 과체중 인상, 둥근 얼굴형, 흐릿한 턱선, 작은 눈, 낮은 이목구비 선명도, 낮은 꾸밈, 어색한 표정
- 용도: 자기유사 월드컵 하단 앵커

## `female_self_52.jpg`

- 최종 외모절대점수: 52
- 백분위 의미: 평균 근처
- 원래 생성 목표: 40점대
- 판정: 이미지 모델 미화 때문에 목표보다 높게 나옴
- 핵심 신호: 평범한 얼굴형, 낮은 꾸밈, 약간 어색한 표정, 보통 이하의 스타일 조화
- 상향 요인: 얼굴 균형, 피부, 조명이 생각보다 안정적
- 용도: 자기유사 월드컵 평균 앵커

## `female_self_73.jpg`

- 최종 외모절대점수: 73
- 백분위 의미: 상위 30% 안쪽 호감형
- 원래 생성 목표: 60점대
- 판정: 이미지 모델 미화 때문에 목표보다 높게 나옴
- 핵심 신호: 균형 잡힌 얼굴, 부드러운 미소, 정돈된 헤어, 안정적인 피부와 조명
- 용도: 자기유사 월드컵 상위 호감형 앵커
