# 자기유사 월드컵 절대점수 기준

이 문서는 자기유사 월드컵에서 사용하는 `외모절대점수`의 의미를 정의한다.

## 핵심 정의

외모절대점수는 호감도 느낌 점수가 아니라 **백분위 점수**다.

예시:

- 90점: 같은 성별·비슷한 연령대 100명 중 상위 10명 수준
- 70점: 100명 중 상위 30명 수준
- 50점: 평균 근처
- 30점: 100명 중 하위 30명 수준
- 10점: 100명 중 하위 10명 수준

따라서 40점은 "나쁘지 않다"가 아니라, 외모 분포에서 하위 40퍼센트 지점이라는 뜻이다.

## 점수 구간

| 점수 | 백분위 의미 | 설명 |
|------|-------------|------|
| 90~100 | 상위 0~10% | 매우 뛰어난 외모. 대부분의 사람이 미남/미녀로 인식 |
| 80~89 | 상위 10~20% | 확실히 눈에 띄는 미남/미녀 |
| 70~79 | 상위 20~30% | 대체로 예쁘다/잘생겼다고 평가받는 호감형 |
| 60~69 | 상위 30~40% | 평균 이상. 호감 요소가 분명함 |
| 50~59 | 평균 근처 | 전체 분포의 중간대 |
| 40~49 | 하위 40~50% | 평균 이하. 미형 신호가 약함 |
| 30~39 | 하위 30~40% | 낮은 호감도 구간. 불리한 외모/관리 요소가 보임 |
| 20~29 | 하위 20~30% | 뚜렷하게 불리한 구간. 자기유사 하한 기준으로 사용 가능 |
| 10~19 | 하위 10~20% | 극단적 하위 구간. 서비스 이미지로는 신중히 사용 |
| 0~9 | 하위 0~10% | 매우 극단적 구간. 실제 서비스 노출은 권장하지 않음 |

## 자기유사 월드컵에서의 사용 원칙

- 사용자 화면에는 점수를 절대 표시하지 않는다.
- 내부 검수/모델링 문서에는 점수를 표시한다.
- 낮은 점수 이미지는 조롱, 희화화, 비인간화 없이 현실적인 기준 이미지로 만든다.
- 낮은 점수 원인을 "더럽다", "괴상하다" 같은 표현으로 만들지 않는다.
- 체형, 얼굴형, 이목구비, 표정, 헤어 정돈, 스타일 조화, 사진 조건을 분리해서 기록한다.
- 사진 품질 때문에 점수가 낮아지는 경우와 얼굴 구조 때문에 점수가 낮아지는 경우를 구분한다.

## 매칭 반영 방식

자기유사 월드컵의 목적은 "내가 몇 점인가"를 사용자에게 알려주는 것이 아니다.
목적은 사용자의 `self_appearance_vector`를 만드는 것이다.

매칭에서는 다음 값을 비교한다.

```text
A의 선호 외모 벡터
vs
B의 자기 외모 벡터
```

그리고 반대 방향도 계산한다.

```text
B의 선호 외모 벡터
vs
A의 자기 외모 벡터
```

최종 외모 궁합은 양방향 유사도를 합산하거나 평균낸다.

## AI 사진 생성 시 주의

이미지 생성 모델은 기본적으로 사람을 예쁘게 보정하는 경향이 있다.
따라서 "30점 인물"이라고 프롬프트에 쓰는 것만으로는 30점대 이미지가 잘 나오지 않는다.

낮은 점수대 기준 이미지를 만들 때는 아래 요소를 더 직접적으로 지정해야 한다.

- 미형 기준에서 불리한 얼굴형
- 작거나 흐릿한 눈
- 낮은 이목구비 선명도
- 어색한 표정
- 낮은 꾸밈과 낮은 스타일 조화
- 과체중 또는 체형상 불리한 인상
- 평범한 실내 조명
- 낮은 사진빨

다만 불쾌함, 혐오감, 조롱 느낌을 만들면 안 된다.
