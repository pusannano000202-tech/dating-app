# 이상형 월드컵 AI 사진

이 폴더에 아래 6장의 AI 생성 사진을 넣어야 합니다.

## 필요한 파일

| 파일명 | 외모 타입 | 설명 |
|--------|-----------|------|
| `cute.jpg` | 귀여운 | 동그란 눈, 애교있는 미소, 발랄한 분위기 |
| `pure.jpg` | 청순한 | 자연스러운 화장, 단아하고 맑은 분위기 |
| `chic.jpg` | 시크한 | 날카로운 눈빛, 세련되고 도도한 분위기 |
| `warm.jpg` | 따뜻한 | 웃는 눈, 친근하고 편안한 분위기 |
| `stylish.jpg` | 스타일리시 | 트렌디한 패션, 개성있는 스타일 |
| `healthy.jpg` | 건강미 | 활기차고 건강한, 생기있는 분위기 |

## 권장 스펙

- 해상도: 768×1024 이상 (세로형 3:4 비율)
- 포맷: JPG (최적화 권장)
- 용량: 장당 500KB 이하 권장
